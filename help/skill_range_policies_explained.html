Skill Range Policies

Skill range policies are a convenient and flexible way to control the restrictions. Policies are cumulative, meaning that the most restrictive will apply.

Name:
 A meaningful name that will be used to identify the policy on the club preferences page.
Description:
 A detailed description of this policy
Skill Range:
 The skill range represents the acceptable skill range for the players. For example
Court:
 This represents the daily limit imposed per court. For a reservation policy, this is the number of reservations that can be made by a user per day. For cancellation policy, this is the number of cancellations that can be made by a user per day.
Court:
 Scheduling policies can be applied selectively to individual courts or can be applied to all courts.
Day of Week:
 Scheduling policies can be applied selectively to specific days or can be applied to all days.
Specify Window:
 Optionally, scheduling windows can be specified. A scheduling window can be created to block off a time range during the day.
Note: When a scheduling window is not specified, the scheduling policy applies for the entire day.
Start Time:
 When the specify window checkbox is checked, the start-time represents the time of day that the policy will be enforced.
End Time:
 When the specify window checkbox is checked, the end-time represents the time of day that policy is no longer enforced.


Examples:
  1. To create a policy restricting players from making any reservation with other players who are ranked more than .25 higher or lower than them:
    Name:
    Skill Range Policy
    Description:
    A cancellation policy restricting users cancellation privileges
    Type:
    Cancellation
    Skill Range:
    .25
    Court:
    All Courts
    Day of Week:
    All Days
    Specify Window:
    < not checked >
    Start Time:
    N/A
    End Time:
    N/A
  2. To create a policy restricting players from making reservations on Saturdays between 11am and 2pm with other players who are ranked more than .5 higher or lower than them:
    Name:
    Prime time Policy
    Description:
    A reservation policy that restricts the number of reservations users to one daily reservation.
    Type:
    Reservation
    Skill Range:
    .5
    Court:
    All Courts
    Day of Week:
    Saturday
    Specify Window:
    < checked >
    Start Time:
    11:00
    End Time:
    14:00
